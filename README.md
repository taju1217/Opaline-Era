# Opaline Era E-Commerce Platform

A modern, elegant e-commerce platform built with React and Node.js.

## Features

- 🛍️ Product browsing with filtering and search
- 🛒 Shopping cart functionality
- ❤️ Wishlist management
- 💳 Checkout process
- 👤 Admin dashboard for product and order management
- 📱 Fully responsive design
- 🎨 Beautiful cloud/white/cream color theme

## Tech Stack

### Frontend
- React 18
- Tailwind CSS
- Framer Motion
- React Router
- Axios
- Swiper.js

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- Multer (file uploads)

## Installation

### Prerequisites
- Node.js (v14 or higher)
- MongoDB
